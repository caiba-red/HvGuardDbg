# HvGuardDbg

本软件是一款基于 Intel 虚拟化技术（VT-x / EPT）的免费调试器保护工具，专为 Windows 平台下的反调试、反检测需求设计，适用于以下场景：

【支持平台】
• 操作系统：Windows 10 / 11（建议使用 Win10 22H2 或更高版本）
• 处理器：Intel 第11/12/13代 Core 系列，开启 VT 支持

【主要功能】
• 虚拟化保护：基于 VMX / EPT 的内存访问控制
• 调试器防护：隐藏硬件断点、软件断点、异常过滤等
• 反反调试：  对抗VMP 反调试相关

【典型使用流程】
1. 调试器启动后，接管调试，添加调试器如 CE DBG 等调试软件

2. 默认启用：
   - 隐藏调试断点
   - 抛异常绕过
   - 调试窗口行为保护

【注意事项】
• 软件驱动不进行卸载 会持续占用VT,不进行调试请退出接管（退出卸载hook）
• 适用于逆向工程、安全测试与驱动级调试等高强度防护场景
• 在使用前请保存好文件 避免软件蓝屏导致丢失等问题
• 请在符合相关法律法规前提下使用本工具

本工具旨在为研究人员提供稳定、安全、高兼容性的调试器防护环境。
